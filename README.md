## 使用说明

- 按文件名或上表链接直接打开对应 Markdown 阅读。
- 新增方案时，在仓库根目录添加 `.md` 文件，并同步更新本 README 的「文档索引」表格。
- `.history` 为编辑器本地历史记录，可忽略，不必纳入版本管理。

## 说明

- 文档描述的是**特定项目约束下的工程实践**，并非通用标准；落地前请结合自家架构与安全要求评估。
- 涉及加密、密钥、网关等章节，请先阅读各文档中的「安全边界」或「设计背景」再决定是否采用。
